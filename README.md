# Peer-n-Play
Peer to peer audio streaming application

Dependencies: vlc media player.

Step 1:
make

Step 2.1:
Run server(only one instance allowed)

Step 2.2:
Put audio files for serving peer in audio/ directory

Step 3:
Run peer

Step 4:
Request file name and enjoy!
